# FoodTracker
A showcase of a Java application that allows a user to keep track of various Ingredients and Recipes.
 - Uses SQL Server and JavaFX

# Table of Contents
* [Prerequisites](#prerequisites)
* [Features](#features)

# Prerequisites

* SQL Server
* Java 1.8
